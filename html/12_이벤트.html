이벤트(Event)

    * 이벤트 : 동작, 행위, (행사)
    -> 브라우저에서 할 수 있는 동작

    ex) click, keydown, keypress, keyup,
      mouseover, mouseon, mouseout, mouseleave,
      drag, change, submit, ....
  
  
  
    HTML 이벤트 종류 참고
  


  * 이벤트 리스너(Event Listener)
  -> 브라우저에서 이벤트가 감지 될 때 까지 대기하다가
  감지가 되었을 때 연결된 기능(함수)를 수행하는 것

  - 이벤트 리스너는 "on이벤트명" 형식으로 작성됨

  ex) onclick, onchange, onsubmit, onkeyup ...



  * 이벤트 핸들러(Event Handler)
  -> 이벤트 리스너에 연결된 기능(함수)으로
  이벤트가 감지 되었을 때 수행할 내용을 작성한 함수(function)

인라인 이벤트 모델(Inline)

    HTML 요소에 
    이벤트리스너="이벤트핸들러"
    형식을 작성하는 방법

    (여태 앞에서 했던 모양...)

    -하나의 요소에 동일한 이벤트 핸들러를 중복해서 작성 X
     (이전에 작성된 이벤트 핸들러를
      새로운 이벤트 핸들러가 덮어쓰기함)

    * 최근에는 사용을 많이 하진 않지만 종종 사용됨

    * 인라인 이벤트 모델 작성 시 this
      -> 이벤트가 발생한 현재 요소

고전 이벤트 모델

    - HTML 요소에 이벤트 관련 코드를 작성하지 않고
      JS에서 DOM 객체에 직접 이벤트 관련 코드를 작성하는 방법

    - 하나의 요소에 동일한 이벤트 핸들러를 중복해서 작성 X

표준 이벤트 모델(요소.addEventListtener())

    - W3C (웹 표준 재정 단체)에서 공식적으로 지정한 이벤트 모델

    -장점 : 한 요소에 중복되는 이벤트 리스너를 만들어 넣어
           여러 이벤트 핸들러를 추가할 수 있다!!
