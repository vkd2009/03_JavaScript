window 내장 객체

    - 브라우저 창 자체를 나타내는 객체

    - 자바스크립트 모든 객체의 최상위 부모

    - window의 자식 객체
      1) DOM (Document Object Model) : Document
         (브라우저에서 내용(페이지) 출력 부분)

      2) BOM (Browser Object Model) : location, history, screen, navigator
         (DOM을 제외한 브라우저 나머지 부분)

시간 지연 함수

      - window에서 제공하는 시간 관련 함수들은
        모두 비동기로 동작

비동기(Asynchronous)란?

- 코드 해석(작업)이 작성 순서대로 수행되지 않고 여러 작업이 동시 수행되는 것 -> 먼저 읽은 코드가 수행이 완료되는 것을 기다리지 않고 바로 다음 코드를 수행하는 것 [참고] 동기 - 앞에 코드가 수행이 종료 되어야 다음 코드 수행

[window.]setTimeout(함수, 지연시간(ms))

      - 지연시간이 지난 후 함수를 1회 수행하도록 하는
        window 객체 내장 함수

[window.]setInterval(함수, 지연시간(ms))

        - 지연시간이 지날 때 마다 함수를 계속 수행하도록 하는
          window 객체 내장 함수
